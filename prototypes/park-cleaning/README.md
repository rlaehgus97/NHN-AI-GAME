# 공원 청소부 프로토타입

공원의 공공기물을 직접 청소하는 1인칭 3D 시뮬레이션 게임 프로토타입입니다.

플레이어는 공원을 돌아다니며 벤치, 안내판, 야외 운동기구를 찾아 물청소해야 합니다.  
각 기물은 여러 청소 구역으로 나뉘어 있어, 한 방향에서만 닦는 것이 아니라 기물의 앞면, 뒷면, 측면을 돌아가며 모두 청소해야 합니다.

## 주요 기능

- 1인칭 시점 이동
- 마우스를 이용한 시점 조작
- 물청소 도구 사용
- 기물별 청소 진행도 표시
- 전체 공원 청소 진행도 표시
- 앞면과 뒷면이 구분된 다중 청소 구역
- 청소 완료 효과와 사운드
- 기물 및 공원 오브젝트 충돌 처리
- 플레이 시간 기록
- 청소 완료 결과 화면
- 즉시 재시작 기능

## 청소 대상

총 3개의 공공기물을 청소할 수 있습니다.

1. 벤치
   - 좌석
   - 등받이 앞면
   - 등받이 뒷면
   - 왼쪽 측판
   - 오른쪽 측판

2. 공원 안내판
   - 안내판 앞면
   - 안내판 뒷면
   - 기둥

3. 야외 운동기구
   - 손잡이
   - 몸체
   - 발판
   - 뒷면 커버

총 12개의 청소 구역을 모두 청소하면 게임이 완료됩니다.

## 조작 방법

| 조작 | 기능 |
|---|---|
| `W`, `A`, `S`, `D` | 이동 |
| `Shift` | 달리기 |
| 마우스 이동 | 시점 조작 |
| 마우스 왼쪽 버튼 길게 누르기 | 청소 |
| `Esc` | 마우스 잠금 해제 및 일시 정지 |

## 실행 방법

별도의 설치나 빌드 과정 없이 실행할 수 있습니다.

1. `index.html` 파일을 다운로드합니다.
2. 인터넷에 연결된 상태에서 `index.html`을 더블클릭합니다.
3. Chrome, Edge 등의 웹 브라우저에서 게임을 실행합니다.
4. `공원으로 들어가기` 버튼을 누르면 게임이 시작됩니다.
