# 왁뿌볼이 되어버리다?! — 기술 검증 프로토타입

후보 아이디어의 핵심 기능과 구현 가능성을 확인하기 위해 제작한 간단한 웹 프로토타입입니다.

현재 최종 게임으로 확정된 버전은 아닙니다.

## 검증 목적

- 3D 구 형태 플레이어의 이동 구현
- WASD 조작 및 대시 구현
- 이동 방향에 따른 구 회전
- 장애물 충돌 처리
- 추격자 AI 구현 가능성 확인
- 내구도 및 게임 오버 흐름 확인

## 실행 방법

1. 해당 폴더를 내려받습니다.
2. `index.html`을 실행합니다.
3. 정상 실행되지 않을 경우 VS Code Live Server를 사용합니다.

## 조작법

- WASD: 이동
- Shift: 대시
- H: 디버그 정보 표시

## 현재 상태

기술 구현 가능성을 확인하기 위한 프로토타입이며, 그래픽과 게임 규칙은 임시 구성입니다.
